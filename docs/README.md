# oc2 Docs

These files document shipped `oc2` behavior for the local-first Bun package:

- `architecture.md`: runtime architecture and deferred HTTP API notes.
- `config.md`: JSONC configuration examples and permission rules.
- `mcp.md`: MCP transport examples and auth-required behavior.
- `smoke.md`: manual release smoke checklist.
- `dependencies.md`: dependency review notes.

The implementation specs remain in `../specs/`.
