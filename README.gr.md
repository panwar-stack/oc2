### Εγκατάσταση

```bash
# YOLO
curl -fsSL https://oc2.ai/install | bash

# Διαχειριστές πακέτων
npm i -g oc2-ai@latest        # ή bun/pnpm/yarn
scoop install oc2             # Windows
choco install oc2             # Windows
brew install anomalyco/tap/oc2 # macOS και Linux (προτείνεται, πάντα ενημερωμένο)
brew install oc2              # macOS και Linux (επίσημος τύπος brew, λιγότερο συχνές ενημερώσεις)
sudo pacman -S oc2            # Arch Linux (Σταθερό)
paru -S oc2-bin               # Arch Linux (Τελευταία έκδοση από AUR)
mise use -g oc2               # Οποιοδήποτε λειτουργικό σύστημα
nix run nixpkgs#oc2           # ή github:anomalyco/opencode με βάση την πιο πρόσφατη αλλαγή από το dev branch
```

> [!TIP]
> Αφαίρεσε παλαιότερες εκδόσεις από τη 0.1.x πριν από την εγκατάσταση.

### Εφαρμογή Desktop (BETA)

Το OC2 είναι επίσης διαθέσιμο ως εφαρμογή. Κατέβασε το απευθείας από τη [σελίδα εκδόσεων](https://github.com/panwar-stack/oc2/releases) ή το [oc2.ai/download](https://oc2.ai/download).

| Πλατφόρμα             | Λήψη                               |
| --------------------- | ---------------------------------- |
| macOS (Apple Silicon) | `oc2-desktop-mac-arm64.dmg`   |
| macOS (Intel)         | `oc2-desktop-mac-x64.dmg`     |
| Windows               | `oc2-desktop-windows-x64.exe` |
| Linux                 | `.deb`, `.rpm`, ή AppImage         |

```bash
# macOS (Homebrew)
brew install --cask oc2-desktop
# Windows (Scoop)
scoop bucket add extras; scoop install extras/oc2-desktop
```

#### Κατάλογος Εγκατάστασης

Το script εγκατάστασης τηρεί την ακόλουθη σειρά προτεραιότητας για τη διαδρομή εγκατάστασης:

1. `$OC2_INSTALL_DIR` - Προσαρμοσμένος κατάλογος εγκατάστασης
2. `$XDG_BIN_DIR` - Διαδρομή συμβατή με τις προδιαγραφές XDG Base Directory
3. `$HOME/bin` - Τυπικός κατάλογος εκτελέσιμων αρχείων χρήστη (εάν υπάρχει ή μπορεί να δημιουργηθεί)
4. `$HOME/.oc2/bin` - Προεπιλεγμένη εφεδρική διαδρομή

```bash
# Παραδείγματα
OC2_INSTALL_DIR=/usr/local/bin curl -fsSL https://oc2.ai/install | bash
XDG_BIN_DIR=$HOME/.local/bin curl -fsSL https://oc2.ai/install | bash
```

### Πράκτορες

Το OC2 περιλαμβάνει δύο ενσωματωμένους πράκτορες μεταξύ των οποίων μπορείτε να εναλλάσσεστε με το πλήκτρο `Tab`.

- **build** - Προεπιλεγμένος πράκτορας με πλήρη πρόσβαση για εργασία πάνω σε κώδικα
- **plan** - Πράκτορας μόνο ανάγνωσης για ανάλυση και εξερεύνηση κώδικα
  - Αρνείται την επεξεργασία αρχείων από προεπιλογή
  - Ζητά άδεια πριν εκτελέσει εντολές bash
  - Ιδανικός για εξερεύνηση άγνωστων αρχείων πηγαίου κώδικα ή σχεδιασμό αλλαγών

Περιλαμβάνεται επίσης ένας **general** υποπράκτορας για σύνθετες αναζητήσεις και πολυβηματικές διεργασίες.
Χρησιμοποιείται εσωτερικά και μπορεί να κληθεί χρησιμοποιώντας `@general` στα μηνύματα.

Οι υποπράκτορες είναι εξειδικευμένοι τύποι πρακτόρων που μπορεί να καλέσει ένας κύριος πράκτορας για μια εργασία. Τα teammates είναι διαφορετικά: είναι παρασκηνιακές child sessions μέσα σε ένα agent team, καθεμία με δικό της όνομα, τύπο πράκτορα, role prompt, εξαρτήσεις, mailbox μηνύματα και προαιρετική έγκριση σχεδίου. Ένα teammate μπορεί να εκτελέσει έναν τύπο υποπράκτορα, αλλά το "teammate" είναι ρόλος συντονισμού ομάδας, όχι λειτουργία πράκτορα.

Μάθετε περισσότερα για τους [πράκτορες](https://oc2.ai/docs/agents).
